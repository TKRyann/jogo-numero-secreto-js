# jogo-numero-secreto-js

# 🎯 Jogo do Número Secreto

Um projeto desenvolvido durante meus estudos de **Lógica de Programação com JavaScript** na **Alura**.

O objetivo do projeto foi construir toda a lógica do jogo utilizando JavaScript, praticando conceitos fundamentais da linguagem. O HTML e o CSS já estavam prontos, e meu foco foi desenvolver toda a interação e funcionamento do jogo.

---

## 📖 Sobre o projeto

O jogo consiste em adivinhar um número secreto gerado aleatoriamente entre **1 e 10**.

A cada tentativa, o sistema informa se o número secreto é **maior** ou **menor** que o número escolhido pelo jogador. Quando o usuário acerta, o jogo exibe a quantidade de tentativas utilizadas e permite iniciar uma nova partida.

Além disso, o projeto evita repetir números já sorteados até que todos os números possíveis sejam utilizados.

---

## 🚀 Funcionalidades

* ✅ Geração de número aleatório
* ✅ Validação do chute do jogador
* ✅ Dicas indicando se o número é maior ou menor
* ✅ Contador de tentativas
* ✅ Reinício da partida
* ✅ Controle para evitar repetição dos números sorteados
* ✅ Leitura das mensagens utilizando a biblioteca ResponsiveVoice
